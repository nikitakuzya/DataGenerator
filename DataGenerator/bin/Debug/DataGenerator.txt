	Генератор/редактор сообщений МПСА

	Поддерживается работа с таблицами OIP, PlaceOIP, Sysobject, Object, Message, Timer, PlaceTimer
	Поддерживается работа с базами версий 6 и 7. С ранними версиями работа возможна, но могут быть проблемы.
	Настройки соединения с БД берутся из реестра.
	Файлы CSV, с которыми работает программа совместимы с форматом используемым программой DataConvert

	Функции:

	Генерация сообщений для таблиц из информационного файла и файла сообщений. 

При запуске производится автоматический поиск файлов в папке программы и ее вложенных папках, если файлы имеют тип *xls* и в имени есть маркеры _MSG_ или _IO_, то данные файлы будут использоваться в качестве файлов сообщений и информационного соответственно. Пути к файлам можно указать и вручную.
Для генерации соединение с базой не требуется. Выбор типа МПСА имеет смысл только в режиме генерации.

Версия базы < 7
- OIP генерация Place идет с листа инф файла "ИП" столбец "Видеокадр"
Версия базы = 7
- OIP генерация Place идет с листа инф файла "ИП" столбец "Принадлежность"
- OIP, Timer добавлен столбец PLC 
- Object добавлены столбцы SoundMES, SoundID, Sound

При генерации OIP есть опция автоматически заполнять пропуски данных для столбца Place - будет просвоен 0.


	Заполнение таблиц из файлов CSV

Файлы одноименные с таблицами нужно положить в папку Import. Кавычки (") будут удалены при чтении. Первая строка с заголовками, пустые строки пропускаются. Количество столбцов должно быть не меньше количества столбцов в целевой версии БД
 
	Проверка данных в таблицах на пропуски данных

Соответствующие чекбоксы будут покрашены зелеными или красным.

	Экспорт таблиц в файлы CSV

Строковые данные заключаются в кавычки ("). Названия параметров преобразуются в верхний регистр.
Файлы экспорты помещаются в папку Export_имя информационного файла, а также в папку Import, откуда могут быть сразу считаны.

	Редактор базы данных

Режим редактора работает независимо от других функций. Позволяет просматривать таблицы из базы в интерактивном режиме. Поддерживается изменение данных с подтверждением нажатием кнопки "Записать изменения".
